# ExpressGenGen
ExpressGenGen an Express Code Generator Generator

### Compiling

* CMakeLists.txt - to use with cmake
#### For Compiling Under MinGW
```
mkdir build 
cd build 
cmake -G "MinGW Makefiles" ..
make
```
* xgengen.pro, boost.pri, deployment.pri - to use with qmake
